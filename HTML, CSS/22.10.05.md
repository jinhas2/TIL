HTML : 콘텐츠를 작성하는 역할, 구조를 잡는 역할.

- vs code 확장 프로그램 설치

live server → 파일을 저장하면 새로고침 안해도 브라우저에 변경사항이 나타남. 

open in browser → 마우스 오른쪽 클릭해서 에디터에서 바로 브라우저를 열 수 있음. 

*fn키 + 화살표 → 빠르게 태그 뒤로 이동 가능. 

*ctrl + /키 → 주석처리
